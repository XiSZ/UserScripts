# UserScripts

A collection of user scripts and utilities.

## Contents

- **Global/** - Global utility scripts

  - `url-scanner.js` - URL scanning utility

- **Twitch/** - Twitch platform scripts
  - `picture-in-picture.js` - Picture-in-picture functionality for Twitch streams | Credit: @benjammin4dayz

## Usage

These scripts can be used as browser userscripts or standalone utilities depending on their implementation.
